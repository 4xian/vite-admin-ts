# vite-admin-ts

🎉 基于 Vite 2.x + Vue 3.x(setup) + antd-design-vue + TS 的后台管理系统

## 开发

```bash
# 克隆项目
git clone https://github.com/4xian/vite-admin-ts.git

# 进入项目目录
cd vite-admin-ts

# 安装依赖
npm install

# taobao 源解决下载速度慢的问题
npm config set registry https://registry.npm.taobao.org/

# 启动服务
npm run dev
```

## 发布

```bash
# 构建生产环境
npm run build
```

## 其它

```bash
# eslint代码格式检查
npm run lint:js

# stylelint代码样式检查
npm run lint:css
```
